== Description ==
Extract GPS data from Yet another photoblog plugin images and creates meta data for Geotag plugin.
Geotag plugin is a great geotaggin solution, but it doesn't read exifs from YAPB images.
This plugin reads GPS data when you edit the post, and adds the metadata as if you entered in the Geotag fields.

= Requriments =
You need 
* <a href="http://wordpress.org/extend/plugins/yet-another-photoblog/">Yet another photoblog plugin</a>
* <a href="http://wordpress.org/extend/plugins/geotag/">Geotag plugin</a>


== Installation ==
* Download and activation.
* If you want add geotag data to previous post edit them. You just need to "update" making no changes.
